# 📱 Mobile Price Prediction using Machine Learning

## 📌 Project Overview

This project predicts the price of a mobile phone based on its specifications using the Linear Regression algorithm. The model learns from historical mobile data and predicts the expected price of a new mobile phone.

This project was developed as part of my AIML Internship.

---

## 🎯 Objective

The main objective of this project is to build a Machine Learning model that can accurately predict mobile phone prices using different hardware specifications.

---

## 📂 Dataset

The dataset contains **500 mobile phone records** with the following features:

- RAM (GB)
- Storage (GB)
- Battery Capacity (mAh)
- Camera (MP)
- Display Size (Inches)
- Processor Score

**Target Variable:**
- Price (₹)

---

## 📋 Project Workflow

1. Import required libraries
2. Load the dataset
3. Explore the dataset
4. Check missing values
5. Analyze dataset statistics
6. Split data into training and testing sets
7. Train the Linear Regression model
8. Predict mobile prices
9. Evaluate the model using MAE and R² Score
10. Test the model with new mobile specifications

---

## 📈 Model Performance

- **Training Data:** 400 records
- **Testing Data:** 100 records
- **Mean Absolute Error (MAE):** 1596.28
- **R² Score:** 0.98755 (98.76%)

The high R² Score indicates that the model predicts mobile prices with very good accuracy.
